# 发布版说明

这个文件夹里的版本是给国内访问优化过的。

## 为什么会更快

- 不再使用 React、Tailwind、Babel 的国外 CDN。
- 页面样式和交互都写在 `index.html` 里。
- 打开页面时不需要下载额外框架文件。
- 上传到服务器后，会自动启用简单缓存，第二次打开更快。

## 怎么给爸爸妈妈用

最简单做法：把整个 `发布版` 文件夹上传到国内静态网站托管服务。

可以选：

- 腾讯云 COS 静态网站
- 阿里云 OSS 静态网站
- 七牛云 Kodo
- 又拍云
- Gitee Pages
- Cloudflare Pages
- Netlify / Vercel

如果主要给国内家人访问，优先选国内服务，例如腾讯云 COS、阿里云 OSS、七牛云或又拍云。

## 上传哪些文件

上传这个文件夹里的全部文件：

- `index.html`
- `manifest.webmanifest`
- `service-worker.js`

## 手机上怎么放到桌面

网站上传后，爸爸妈妈用手机浏览器打开网址：

- iPhone：点分享按钮，选择“添加到主屏幕”。
- 安卓：浏览器菜单里选择“添加到桌面”或“安装应用”。

添加后，看起来就像手机上的一个小应用。
