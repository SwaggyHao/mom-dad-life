# 给爸爸妈妈的生活指南

郝煜伽给爸爸妈妈整理的家庭生活指南，使用 React + Tailwind CSS，妈妈健康提醒数据保存在浏览器 localStorage，不需要后端。

## 运行方法

方法一：直接打开

双击 `index.html`，用浏览器打开即可。

方法二：本地服务器打开

```bash
cd /Users/haoyujia/Downloads/爸妈生活助手
node server.mjs
```

然后在浏览器打开：

```text
http://localhost:5173
```

## 修改联系号码

打开 `index.html`，搜索：

```text
tel:10000000000
```

把这个占位号码改成真实电话号码即可。

## 数据保存

爸爸和妈妈的打卡会自动保存到当前浏览器的 localStorage。清除浏览器数据后，记录也会被清除。

## 页面内容

- 首页
- 知识普及
- 家电安心指南
- 妈妈健康提醒
- 妈妈饮食建议
- 爸爸健康提醒
- 妈妈运动建议
- 肉蛋奶水果蔬菜指南
